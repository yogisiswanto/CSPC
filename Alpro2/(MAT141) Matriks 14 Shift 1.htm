(MAT141) Matriks 14 Shift 1


Pembuat Soal: Tim Pemrograman

Batas Waktu Eksekusi 5 Detik
Batas Memori 1 MB

Piyu adalah seorang programmer, dia ingin membuat suatu program tentang matriks. Program yang ingin dibuatnya dapat menukar baris-baris dalam matriksnya dan dapat menggeser suatu baris sebanyak yang piyu inginkan. Ukuran matriksnya ditentukan oleh piyu (baris, kolom).

Masukan :
B, banyaknya baris matriks piyu. 0 < B < 100
K, banyaknya kolom matriks piyu. 0 < K < 100
Matriks ukuran B x K
N, banyaknya perubahan 0 < N < 50
pilihan x y, string int int diulangi sebanyak N 
    Pilihan, berupa string dengan masukan tukar, kanan atau kiri.
        Jika pilihan adalah tukar, maka program akan meminta masukan x dan y, x adalah baris yang akan ditukar dengan y.
        Jika pilihan adalah kanan, maka program akan meminta masukan x dan y, x adalah baris yang isinya akan digeser ke kanan sebanyak y kali.
        Jika pilihan adalah kiri, maka program akan meminta masukan x dan y, x adalah baris yang isinya akan digeser ke kiri sebanyak y kali.
Keluaran :
Matriks yang telah dirubah.
